Navegadores Web

Un Navegador web ( en inglés, web browser) es un software, aplicación o programa que permite el acceso a la Web, Interpretandó la Informaciónde distintos tipos de archivos y sitios web para que estos puedan ser vistos.

La funcionalidad básica de un navegador web es permitir la visualizacicón de documentos texto, posiblemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari:Safari es un navegador web de código cerrado desarrollado por Aplle Inc. Esta disponible para macOS, IPadOS e iOS.

Mozilla Firefox es un navegador web libre y de código abierto desarrollado para Linux, Android, IOS, macOS y Microsoft WIndows Coordinado por la Corporación Mozilla y la Fundación Mozilla.

Google Chrome: Google Chrome es un navegador web de código cerrado desarrollado por Google, aunque derivado de proyectos de código abierto
